COMPLETE DATA GUIDE: NON-SYNTHETIC DATA INVENTORY AND ACQUISITION
=================================================================

Last updated: 2026-02-27
Scope: Every non-synthetic dataset currently under Data_Sources/, what it contains, and how it was acquired.


1) EXTERNAL ACQUISITION CHANNELS
================================

NOAA ISD / Global Hourly (direct files):
- Access: Public file download (no key required for ISD history/global hourly files already downloaded).
- Used for: Station metadata and hourly weather observations (Dulles station 72403093738).

NOAA CDO API:
- Access: NOAA token required.
- Token signup: https://www.ncdc.noaa.gov/cdo-web/token
- Used for: Daily station observations for Dulles/Ashburn area.

EIA Open Data API v2:
- Access: EIA API key required.
- Key signup: https://www.eia.gov/opendata/register.php
- Used for: PJM hourly demand, forecast, interchange, net generation, generation by fuel; Virginia state electricity/emissions series.

Google BigQuery public datasets:
- Access: Google Cloud project + BigQuery API + billing enabled.
- Console: https://console.cloud.google.com/bigquery
- Used for: Google ClusterData 2019 and PowerData 2019 extracts; NOAA GSOD query workflows in legacy scripts.

EPA eGRID:
- Access: Public download.
- Used for: eGRID detailed/metric XLSX reference files and release notes.

Alibaba cluster data:
- Access: Public metadata/docs and survey-gated archive links.
- Used for: External workload trace archives and metadata.


2) DATA_SOURCES ROOT (RAW OR IMPORTED FILES)
============================================

- ashburn_va_temperature_2019.csv
  Contains: hourly timestamp and temperature_c for 2019.
  Acquired: legacy curated weather table already present in repository (NOAA-derived station weather series).

- ashburn_va_temperature_2024.csv
  Contains: NOAA-style daily records (date, datatype, station, attributes, value) for 2024.
  Acquired: NOAA CDO API pull (station GHCND:USW00093738, Dulles).

- google_cluster_utilization_2019.csv
  Contains: real_timestamp, hour_of_day, avg_cpu_utilization, num_tasks_sampled.
  Acquired: extracted from Google ClusterData 2019 public dataset (BigQuery/public trace workflow).

- noaa_dulles_daily_2015_2024.csv
  Contains: daily Dulles weather features (avg/max/min temp, wind, precipitation, humidity, snow) with year/month.
  Acquired: NOAA CDO API via legacy downloader workflow.

- noaa_gsod_dulles_2019.csv
  Contains: GSOD daily weather fields (temp, dewp, max, min, prcp, sndp) for Dulles 2019.
  Acquired: NOAA GSOD query workflow (legacy BigQuery/public GSOD extraction).

- pjm_hourly_demand_2019_2024_eia.csv
  Contains: datetime_utc, respondent, demand_mwh.
  Acquired: EIA API v2 endpoint electricity/rto/region-data/data with respondent=PJM, type=D.

- pjm_demand_forecast_2019_2024_eia.csv
  Contains: period, respondent metadata, type=DF, value (MWh).
  Acquired: EIA API v2 endpoint electricity/rto/region-data/data with respondent=PJM, type=DF.

- pjm_interchange_2019_2024_eia.csv
  Contains: period, respondent metadata, type=TI, value (MWh).
  Acquired: EIA API v2 endpoint electricity/rto/region-data/data with respondent=PJM, type=TI.

- pjm_net_generation_2019_2024_eia.csv
  Contains: period, respondent metadata, type=NG, value (MWh).
  Acquired: EIA API v2 endpoint electricity/rto/region-data/data with respondent=PJM, type=NG.

- pjm_generation_by_fuel_2019_2024_eia.csv
  Contains: period, respondent metadata, fueltype, value (MWh).
  Acquired: EIA API v2 endpoint electricity/rto/fuel-type-data/data with respondent=PJM and fuel filters.

- virginia_co2_emissions_2015_2023.csv
  Contains: annual/state emissions identifiers (sector, fuel, state metadata).
  Acquired: EIA open data state-level emissions series (saved CSV export).

- virginia_electricity_consumption_2015_2024.csv
  Contains: monthly state electricity consumption identifiers (period, state, sector).
  Acquired: EIA open data state-level consumption series (saved CSV export).

- virginia_generation_by_fuel_2015_2024.csv
  Contains: monthly Virginia generation metadata by fuel and sector.
  Acquired: EIA open data state-level generation-by-fuel series (saved CSV export).

- virginia_renewable_generation_2015_2024.csv
  Contains: monthly Virginia renewable generation metadata by fuel and sector.
  Acquired: EIA open data state-level renewable generation series (saved CSV export).

- EIA923_Schedules_2_3_4_5_M_11_2025_21JAN2026.xlsx
  Contains: EIA-923 monthly generation/fuel schedules workbook.
  Acquired: manual public download from EIA Form 923 release files.

- epa_temperature_loudoun_2019.csv
  Contains: empty file (0 rows).
  Acquired: incomplete/failed export artifact; not used.

- datacenter_constants.json
  Contains: fixed project constants/assumptions.
  Acquired: project-authored local configuration (not externally downloaded data).

- data_quality_report.txt
  Contains: legacy cleaning report output.
  Acquired: generated by local quality/cleaning script runs.

9) REPRODUCIBLE BUILD PATH (NON-SYNTHETIC PIPELINE)
===================================================

Step 1: Acquire raw external data
- EIA API pulls (PJM and Virginia series)
- NOAA direct downloads and/or NOAA CDO API pulls
- Google cluster/power extracts (BigQuery workflow)
- EPA eGRID and Alibaba archives/docs (direct download)

Step 2: Create cleaned non-synthetic tables
- Run local cleaning scripts (existing project cleaning workflow).
- Run: Data_Sources/scripts/build_additional_cleaned_datasets.py
  Outputs:
  - cleaned/pjm_exogenous_hourly_2019_2024_cleaned.csv
  - cleaned/noaa_global_hourly_dulles_2019_2024_cleaned.csv

Step 3: Build model-ready non-synthetic tables and reports
- Run: Data_Sources/scripts/data_audit_and_organize.py
  Outputs:
  - reports/*.csv and reports/DATA_AUDIT_SUMMARY.md
  - model_ready/stage*_*.csv and final_training_table_*.csv
