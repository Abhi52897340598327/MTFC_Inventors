PHASE 2: INSTALL GOOGLE CLOUD SDK (10-15 min) 💻
-------------------------------------------------
This is done on your local computer.

→ Download SDK installer:
  https://cloud.google.com/sdk/docs/install-windows

→ Install and authenticate:
  1. Run GoogleCloudSDKInstaller.exe
  2. Close and reopen PowerShell
  3. Run: gcloud auth application-default login
  4. Run: gcloud config set project YOUR-PROJECT-ID

→ Test installation:
  gcloud --version
  (Should show version info, not "command not found")

When done: ✓ Check off all Phase 2 boxes in the checklist


PHASE 3: DOWNLOAD DATA (5-30 min depending on mode) 📥
-------------------------------------------------------
This is where you get the actual power consumption data!

STEP 1: Update scripts with your Project ID
  • Open: Model_Files/test_bigquery_connection.py
  • Line 89: Replace "YOUR_PROJECT_ID_HERE" with your actual ID
  • Save file
  
  • Open: Model_Files/download_bigquery_data.py  
  • Line 29: Replace "YOUR_PROJECT_ID_HERE" with your actual ID
  • Save file

STEP 2: Test connection (IMPORTANT - do this first!)
  → Run in PowerShell:
    .venv\Scripts\python.exe Model_Files\test_bigquery_connection.py
  
  → Expected: "SUCCESS! BigQuery connection is working perfectly! ✅"
  → If errors, see troubleshooting section

STEP 3: Test with single PDU (quick test before full download)
  → Run:
    .venv\Scripts\python.exe Model_Files\download_bigquery_data.py --test
  
  → Should download ~1000 rows in ~10 seconds
  → Creates: Data_Sources/google_power_data_2019_sample.csv
  → Confirms everything works before big download

STEP 4: Full download (choose one)
  
  Option A - Sample Mode (RECOMMENDED FOR TESTING)
    → Good for: Model development, testing, rapid iteration
    → Time: ~2-3 minutes
    → Size: ~10-20 MB
    → Command:
      .venv\Scripts\python.exe Model_Files\download_bigquery_data.py --sample
  
  Option B - Complete Dataset (FOR FINAL MODEL)
    → Good for: Final model training, maximum accuracy
    → Time: ~20-30 minutes
    → Size: ~50-100 MB
    → Command:
      .venv\Scripts\python.exe Model_Files\download_bigquery_data.py --full
  
  💡 TIP: Start with --sample mode to test everything works,
          then run --full mode when you're ready for final model.

STEP 5: Validate data quality
  → Run:
    .venv\Scripts\python.exe Model_Files\validate_data.py
