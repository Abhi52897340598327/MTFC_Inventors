STEP 3: DOWNLOAD BIGQUERY DATA (1 hour first time)
---------------------------------------------------
□ Setup Google Cloud:
  1. Visit: https://console.cloud.google.com/
  2. Create new project (any name)
  3. Enable BigQuery API
  4. Enable billing (FREE TIER covers this!)

□ Download power data:
  Option A - Web Console (easiest):
    1. Open BigQuery SQL workspace
    2. Paste query from complete_data_guide.txt
    3. Click Run
    4. Export results to CSV
    5. Save as: google_power_data_2019.csv
  
  Option B - Python script:
    1. Install: pip install google-cloud-bigquery
    2. Authenticate: gcloud auth login
    3. Open: Model_Files/download_bigquery_data.py
    4. Set PROJECT_ID
    5. Run: python download_bigquery_data.py

Files downloaded:
  □ google_power_data_2019_raw.csv
  □ google_power_data_2019_hourly.csv
  □ machine_to_pdu_mapping.csv
